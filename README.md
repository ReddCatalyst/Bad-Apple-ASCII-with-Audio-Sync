### Sebelum menjalankan, wajib install ini dulu loh ya!:

1. **[Python 3.x](https://www.python.org/downloads/)** bebas yang penting versi 3 (**Penting: Pastikan centang opsi "Add Python to PATH"**).

```bash

```

3. Jika sudah ada Python, lalu ketik ini di CMD untuk menginstall library-nya:
```bash

```

## üöÄ Cara Menjalankan

### 1. Masukkan Assets

Pastikan file video dan musik sudah ada di folder `assets/`:

* `assets/bad_apple.mp4`
* `assets/bad_apple.mp3`

### 2. Jalankan Program

Ada 2 cara jalaninnya (mending langsung ke opsi A aja biar ga pusing jir)

* **Opsi A (Paling Gampang):**
Cukup klik dua kali pada file `bad_apple.bat` di folder utama.
* **Opsi B (Lewat Terminal):**
Buka CMD di folder utama proyek, lalu ketik:
```bash
python src/bad_apple.py

```
